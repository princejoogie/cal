## Developing

- Clone the repository

  ```bash
  git clone https://github.com/princejoogie/cal.git
  ```

### cal-js

- Run

  ```bash
  yarn install
  yarn start
  ```

- Build

  ```bash
  yarn build
  # or build for specific targets
  yarn build:{win|mac|linux} # eg yarn build:win
  ```

### cal-rust

- Run

  ```bash
  Cargo run
  ```

- Build (optimized)

  ```bash
  cargo build --release
  ```
