## Developing

```bash
git clone https://github.com/princejoogie/cal.git
```

### C++ implementation

```bash
# Run
make run

# Build
make
```

### JavaScript implementation

```bash
# Run
yarn start

# Build
yarn build

# Targeted build
yarn build:(win|mac|linux)
```

### Rust implementation

```bash
# Run
Cargo run

# Build
cargo build --release
```
